# Setup

**Prerequisites:**
1. npm
2. yarn
3. glass of water. remember to drink some. its good for you

**Installing and Running**:
1. Clone the repo
```
git clone git@github.com:MaritzSTL/data-reference.git
```
2. Install dependencies
```
cd data-reference
yarn install
```
3. Serve it
```
yarn run serve
```
Navigate to http://localhost:4000/graphql
